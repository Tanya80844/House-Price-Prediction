# 🏠 House Price Prediction using Multiple Linear Regression

---

## 📌 Problem Statement
The aim of this project is to build a Machine Learning model that predicts house prices based on multiple features using Multiple Linear Regression.

---

## 📊 Dataset
The dataset used in this project contains information about houses such as:

- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Price (Target Variable)

---

## ⚙️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🔍 Project Workflow
1. Data Loading
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Feature Scaling
6. Model Training
7. Prediction
8. Model Evaluation
9. Visualization

---

## 📈 Model Performance
The model was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

The visualization of Actual vs Predicted values shows good prediction accuracy.

---

## 📉 Visualization
Actual vs Predicted house price comparison graph was used to evaluate model performance.

---

## 🏁 Conclusion
Multiple Linear Regression successfully predicted house prices with good accuracy.  
The project demonstrates how regression models can be applied to real-world prediction problems.
